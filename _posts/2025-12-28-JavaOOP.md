## 三、构造器
### 1.定义
构造器实际上是一个方法，作用是可以对实例进行初始化 
```java
 public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
```
使用示例：
```java
Person person = new Person("xiaoming",20);
        System.out.println(person.name);
```
一个类中即使什么都没写，也会有一个无参构造器，new一个类（即创造一个对象的过程）其实是调用构造器的过程。
### 2.分类
构造器分为有参构造器和无参构造器。当定义了一个有参构造器，但又需要使用无参构造器的时候需要重新定义一个无参构造器。
### 3.如何得到一个构造器
1）方法名要与类名完全一样  
2）没有返回值
快捷键：Alt+Insert,Shift
